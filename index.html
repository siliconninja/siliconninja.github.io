Projects and Development

RHS Schedule
[PBL] [JS]

Check the schedule, absences, and more for RHS, right on your Pebble. You can also see the time before each period ends and the local weather. It is developed in Pebble.js, which consists of JavaScript bindings for the Pebble SDK written in C.

Download App / Source

Rhythm Jumper
[WIN] [VB.NET]

A fun and challenging music game with 3 different levels, exciting electronic music, and high scores! Also includes secret easter eggs. It was written in Visual Basic .NET as a final project for a Computer Science class.

Source

Total School Days
[WEB] [JSON API] [CDB]

The burning question, answered. Includes a countdown, progress bar, and school day updater script. Originally developed for RHS from 2015-2017. Utilizes the Bootstrap web development platform and includes a JSON API for other applications, including RHS Schedule, to read the total school day count.

Source

Redirector
[WEB] [NGINX]

Add custom thumbnail images to Google Classroom posts and social media posts (alpha). Uses a timeout to delay loading of the final webpage so the link doesn't display the page as a thumbnail. Give your links some style!

Source

Tech Club @ School
[DOC] [PY] [CYSEC] [HTML] [CSS]

I have taught RHS students how to code a simple Mad Libs game in Python, create a website in HTML, and cybersecurity through CTF competitions! These are the resources I have gathered, developed, and used. They are developed specifically for high school/middle school tech clubs, but can be used by anyone.

This Map In History
[WEB] [RUBY]

A website that allows you to view historical events of every state in the United States by day of the year, like This Day In History. Utilizes Nokogiri to scrape events; Sinatra, Ruby, and ERB for the frontend and backend. Source for events: OnThisDay.com (before the redesign). Created as a project at the Flatiron School with Clink123 and dankong.

Rename My Playlist
[SRC] [N.JS]

A webapp where you can easily rename a Spotify playlist easily on any device (built when the feature did not exist within the Spotify app for iOS). It uses the Spotify Web API and Node.js for the backend. The frontend has a simple interface which uses Bootstrap.

Source
